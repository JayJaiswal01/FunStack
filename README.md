# 🎮 FunStack 🎯

### Stacked with Fun, Powered by Full Stack

## 📌 Overview

FunStack is a full-stack web application that combines multiple interactive mini-games and utility tools into a single platform with modern UI and 3D effects.

---

## 🚀 Features

* 🎯 Number Guessing Game
* 🎡 Spin Wheel / Random Picker
* 🎲 Dice Battle Game
* 🧠 Quick Quiz Game
* 😂 Random Task Generator

---

## 🎮 Game Modes

* 👤 User vs Computer
* 👥 User vs User (Local Multiplayer)

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS
* Framer Motion

### Backend:

* Java
* Spring Boot

### Database:

* MySQL

---

## 📁 Project Structure

```
FunStack/
├── frontend/
├── backend/
```

---

## ⚙️ How to Run

### Frontend:

```
cd frontend
npm install
npm run dev
```

### Backend:

```
cd backend
mvn spring-boot:run
```

---

## 🎯 Objective

To build an interactive and engaging full-stack application demonstrating UI/UX design, game logic, and backend integration.
